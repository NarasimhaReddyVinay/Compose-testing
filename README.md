# Expense Tracker 📊

**Expense Tracker** is a clean and simple app built for Android. It helps users manage their personal finances by tracking income and expenses with a modern, intuitive interface.

## Core Features
*   **Transaction Management**: Add, edit, delete, and view details of your income and expenses.
*   **Dashboard Overview**: View your total balance, total income, and total expenses at a glance.
*   **Filtering**: Filter transactions by type (All, Income, or Expense) to better understand your spending habits.
*   **Dark Mode Support**: Fully supports system-wide dark mode for comfortable use at night.
*   **Data Export**: Export your transaction history to a CSV file for external record-keeping.
*   **Share as Image**: Share specific transaction details as an image with others.

## Tech Stack
The app is built using modern Android development tools and best practices:
*   **Language**: Kotlin
*   **Architecture**: MVVM (Model-View-ViewModel)
*   **Database**: Room (SQLite) for local data persistence.
*   **Dependency Injection**: Hilt (Dagger) for clean and testable code.
*   **UI**: XML with ViewBinding and Material Components.
*   **Async**: Kotlin Coroutines and Flow for reactive data handling.
*   **Navigation**: Jetpack Navigation Component for consistent fragment transitions.
*   **Storage**: DataStore for lightweight preference storage (like theme mode).

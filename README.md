# Expense Tracker (Pure Compose) 📊

**Expense Tracker** is a modern, high-performance personal finance app built **100% with Jetpack Compose**. It demonstrates advanced Android development techniques including custom graphics, reactive state management, and declarative UI.

## Core Features
*   **Pure Compose UI**: Built entirely without XML layouts or Fragments.
*   **Interactive Analytics**: Custom-drawn pie chart using the Compose `Canvas` API with entry animations.
*   **Dynamic Dashboard**: Real-time balance tracking and transaction history.
*   **Single Activity Architecture**: Powered by `androidx.navigation.compose` for seamless transitions.
*   **Material 3**: Fully implements the latest Material Design guidelines.
*   **Dark Mode**: Native support for system-wide dark theme.

## Tech Stack
*   **UI Framework**: Jetpack Compose (100%)
*   **Navigation**: Compose Navigation
*   **Database**: Room (SQLite)
*   **Dependency Injection**: Hilt
*   **State Management**: Flow, StateFlow, and Compose State
*   **Testing**: Compose UI Testing with `createComposeRule` and MockK.
